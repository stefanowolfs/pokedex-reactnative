# My Pokedex with react-native

## Installation

```bash
# add node_modules dependencies
$ yarn

# add Pod dependencies
$ npx pod-install ios
```

## Running the app

```bash
# development
$ npx react-native run-ios
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
